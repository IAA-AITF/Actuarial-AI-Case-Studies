# Case Study Templates

Professional templates for actuaries writing AI-related case studies. Includes both Jupyter Notebook and RMarkdown versions to support Python and R users.

For full contribution guidelines, see [CONTRIBUTING.md](../CONTRIBUTING.md).

---

## Contents

- **`jupyter-templates/`**
  - `iris_case_study_template_simple.ipynb` — Simple template using Markdown only
  - `iris_case_study_template_advanced.ipynb` — Advanced template using Markdown and HTML elements

- **`rmarkdown-templates/`**
  - `iris_case_study_template.Rmd` — RMarkdown template
  - `iris_case_study_template.md` — Markdown export for preview on GitHub

---

## How to Use

1. Navigate to the folder for your preferred language:
   - `jupyter-templates/` for Python users
   - `rmarkdown-templates/` for R users
2. Open the template file in JupyterLab/Notebook or RStudio.
3. Replace the placeholder text and code with your case study content.
4. Export as HTML (Jupyter) or Knit-to-HTML (RMarkdown + RStudio) for GitHub-ready rendering.
